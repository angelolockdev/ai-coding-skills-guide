Le workflow

Une seule ligne a retenir

Chaque etape produit un livrable verifiable qui alimente la suivante.

Clarifier → Specifier → Construire → Prouver → Designer → Polir → Auditer
Les 5 skills

Chaque skill a un role precis

Ne demandez pas a l'agent de tout faire d'un coup. Donnez-lui un role, une procedure et des criteres verifiables.

01
🔥

Grill me

Interroge l'utilisateur sans relache sur un plan ou un design avant de coder. Elimine les ambiguites produit pour viser le one-shot.

Clarifier Spec Acceptance
npx skills@latest add mattpocock/skills --skill grill-me
02
⚙

APEX

Analyze, Plan, Execute, Examine. L'agent teste son propre code et fournit des screenshots comme preuve de fonctionnement.

Implementer Tester Screenshot
npx skills@latest add melvynx/aiblueprint --skill apex
03
🎨

Impeccable

Corrige le slop visuel: typographie, contraste, responsive, espacements. 23 sous-skills qui rendent le design intentionnel.

Design Responsive No-slop
npx impeccable install
04
✨

Make Interface Feel Better

Polir les details d'une interface deja fonctionnelle: alignements optiques, transitions, bordures, stabilite des nombres.

Polish Micro-UX Details
npx skills add jakubkrehel/make-interfaces-feel-better
05
☢

Thermo-Nuclear Review

Audit sans pitie de la maintenabilite: code spaghetti, abstractions magiques, tests manquants, dette technique et architecture fragile.

Audit Dette Qualite
