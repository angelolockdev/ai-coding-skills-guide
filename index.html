Les compétences de base

Chaque skill traite une incertitude précise.

Ils ne remplacent pas le jugement. Ils évitent de confondre la découverte, l'exécution, le design et l'audit dans une seule instruction.

Le workflow avant les outils.

Une méthode rend le travail d'un agent visible, discutable et vérifiable.

Grill me

Clarifie les décisions produit avant de créer du code coûteux à corriger.

APEX

Planifie, exécute, teste et examine une tranche de travail.

npx skills@latest add melvynx/aiblueprint --skill apex

Impeccable

Donne une direction visuelle cohérente quand l'interface fonctionne mais reste générique.

Feel Better et Thermo-Nuclear

Polissent l'interface et vérifient que le code reste lisible, direct et durable.

Le parcours de travail

Le site se lit comme le travail avance.

Le défilement suit le chemin réel d'une tâche. Chaque étape prépare une décision ou une preuve utile à la suivante.

Ne sautez pas la partie qui évite le rework.

Les cartes se révèlent dans l'ordre où elles deviennent nécessaires. Sur mobile, elles restent visibles dans une lecture linéaire.

Clarifier le problème.

Avant d'ouvrir un fichier, rendez les ambiguïtés visibles. Le but, les contraintes et les non-objectifs deviennent des critères vérifiables.

  • Audience et résultat attenduCe que le changement doit réellement produire.
  • Limites et risquesCe qui ne doit pas bouger sans validation.

Construire une tranche prouvable.

APEX transforme la demande en plan, puis en une implémentation testée. Le résultat utile est une preuve, pas une promesse.

  • Plan fichier par fichierLe périmètre est petit et les dépendances sont visibles.
  • Validation réelleTests, build et comportement vérifié sur la surface utilisateur.

Designer puis auditer.

Le design vient après la validation fonctionnelle. La revue finale cherche les dettes structurelles qui rendent une solution fragile.

  • Direction avant décorationHiérarchie, accessibilité et interactions ont une raison précise.
  • Revue indépendanteLe générateur ne doit pas être son propre juge.

Questions utiles avant de lancer un agent.

Le guide est conçu pour réduire les faux départs sans transformer chaque petite correction en cérémonial.

Faut-il installer les cinq skills ?

Oui pour le workflow complet. Pour une première amélioration, commencez par Grill me et APEX.

Est-ce que cela remplace une revue humaine ?

Non. Les skills rendent la revue plus structurée. Les changements sensibles gardent une validation humaine.

Quel prompt donne de bons résultats ?

Un prompt qui indique le but, le périmètre, les contraintes, le résultat attendu et la manière de le vérifier.

Pourquoi séparer design et finition ?

Impeccable choisit une direction. Feel Better améliore l'exécution : surfaces, rythme, retours tactiles et stabilité.
