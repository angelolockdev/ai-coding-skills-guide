# Prompt : revue de PR

> Passer en revue les changements de la branche par rapport a la branche de base.

## Template

```text
Passe en revue les changements de la branche par rapport a la branche de base.
Priorise : bugs reels, securite, permissions, perte de donnees, regressions,
contrats API, tests manquants et maintenabilite.

Pour chaque constat, fournis : severite, fichier, preuve, scenario d'impact
et correction minimale. Ignore les preferences stylistiques deja couvertes par le lint.
```
