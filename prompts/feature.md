# Prompt : nouvelle fonctionnalite

> A utiliser avec le bloc de contexte et la specification (issue de Grill me).

## Template

```text
A partir du contexte projet et de la specification fournie :
1. analyse le code concerne ;
2. confirme les criteres d'acceptation ;
3. propose un plan par tranches verticales ;
4. implemente la plus petite tranche utile ;
5. ajoute ou adapte les tests ;
6. execute les validations ;
7. montre la preuve du resultat ;
8. continue jusqu'a satisfaire tous les criteres.

Ne reecris pas des zones sans rapport avec la fonctionnalite.
Arrete-toi et demande une decision si un choix produit majeur manque.
```
