# Prompt : re-test des constats corriges

> Re-teste les constats corriges sans supposer qu'ils sont resolus.

## Template

```text
Re-teste les constats corriges [IDs] sans supposer qu'ils sont resolus.

Pour chacun :
- reproduis l'ancien scenario ;
- verifie le comportement attendu ;
- execute les tests de non-regression ;
- controle les contournements evidents.

Classe chaque constat : Corrige / Partiellement corrige / Non corrige /
Non verifiable.

Fournis les preuves et n'ouvre pas de nouveaux constats sans les separer
clairement du re-test.
```
